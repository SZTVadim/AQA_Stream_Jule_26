ГАЙД: КАК ПОЛУЧИТЬ SSH-КЛЮЧ ДЛЯ GITHUB НА MACOS И WINDOWS
ТЕКСТОВЫЙ ФОРМАТ
Часть 1. Как получить SSH-ключ на macOS

Открой Terminal (через Spotlight или Launchpad).

Выполни команду:
ssh-keygen -t ed25519 -C "твой_email_на_GitHub"

Terminal задаст три вопроса:
— Куда сохранить ключ: просто нажми Enter
— Придумать пароль к ключу: нажми Enter
— Повторить пароль: снова Enter

Ключи будут созданы по пути:
~/.ssh/id_ed25519 — приватный ключ
~/.ssh/id_ed25519.pub — публичный ключ

Чтобы посмотреть публичный ключ, выполни команду:
cat ~/.ssh/id_ed25519.pub

Скопируй всю строку, которая начинается с:
ssh-ed25519 AAAA...

Перейди на GitHub:
Profile → Settings → SSH and GPG keys → New SSH key
Вставь туда содержимое файла id_ed25519.pub.

На этом всё — ключ создан и добавлен.

Часть 2. Как получить SSH-ключ на Windows

Установи Git for Windows, если не установлен:
скачать можно на сайте git-scm.com

Запусти Git Bash
(это отдельное приложение, устанавливается вместе с Git).

Выполни команду:
ssh-keygen -t ed25519 -C "твой_email_на_GitHub"

Ответь на вопросы:
— Место сохранения: Enter
— Пароль: Enter
— Повтор: Enter

Ключи будут созданы в папке:
C:\Users\ТВОЁ_ИМЯ.ssh\id_ed25519
C:\Users\ТВОЁ_ИМЯ.ssh\id_ed25519.pub

Чтобы вывести публичный ключ:
cat ~/.ssh/id_ed25519.pub

Скопируй строку вида:
ssh-ed25519 AAAA...

На GitHub зайди в:
Settings → SSH and GPG keys → New SSH key
Вставь туда содержимое публичного ключа.

Проверка работы ключа (и на Mac, и на Windows)

Выполни команду:
ssh -T git@github.com

Если всё правильно, появится сообщение:
Hi <твой логин>! You've successfully authenticated.
